# Write Hub

[![Commits](https://img.shields.io/badge/Commits-25-blue)]()

Write Hub is a full-stack web app to create, edit, and manage posts with images. Built with React, Node.js, Express, MongoDB, and Cloudinary. Features authentication, CRUD operations, and toast notifications.

---

## Features

- User registration & login (JWT auth, cookie-based)  
- Create, edit, delete posts with images  
- Edit posts with image replacement  
- Private routes for authenticated users  
- Responsive UI (Tailwind CSS & React Icons)  
- Toast notifications for success/error  

---

## Tech Stack

- **Frontend:** React, Tailwind CSS, React Router, React Hot Toast  
- **Backend:** Node.js, Express.js, MongoDB (Mongoose)  
- **Auth:** JWT (cookie-based)  
- **File Storage:** Cloudinary
